## ☁️ Project Overview

VIDHURA is a farmer-focused web platform designed to provide agricultural awareness and accessible support resources through a clean, user-friendly interface.

This repository demonstrates the deployment of the VIDHURA website on **Microsoft Azure** using the **Static Website Hosting** feature of Azure Blob Storage.

---

## 🏗️ Cloud Architecture
User
↓
Azure Storage Account
↓
Static Website Hosting ($web container)
↓
Public Endpoint (web.core.windows.net)

## ⚙️ Deployment Process

1️⃣ Created Azure Storage Account  
2️⃣ Enabled Static Website feature  
3️⃣ Configured:
   - `index.html` (Index document)
   - `404.html` (Error document)  
4️⃣ Uploaded project files to `$web` container  
5️⃣ Accessed Primary Endpoint URL
